<h1 align="center">E2E Testing</h1>

## Prerequisites:

### Install golangci-lint
```bash
go install github.com/golangci/golangci-lint/cmd/golangci-lint@latest
golangci-lint version
golangci-lint help linters
```

### Execute golangci-lint
```bash
golangci-lint run
```
